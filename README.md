# SmartCal — 语音智能日历

SmartCal 是一款基于 Electron 的桌面端语音交互日历工具。用户通过**语音**即可完成日历事件的添加、删除、修改等操作。

视频演示链接：https://www.bilibili.com/video/BV1p4VQ6XE4S/?vd_source=e8b2568138d8df734f6990a4c32012ce

---

## 一、功能概述

### 1.1 核心功能

| 功能             | 说明                                                  |
| ---------------- | ----------------------------------------------------- |
| **语音添加事件** | 说出事件名称和时间，系统自动识别并添加到日历          |
| **语音删除事件** | 说出要删除的事件名称，系统匹配后标记为"已完成"        |
| **语音修改事件** | 说出修改前和修改后的事件信息，系统自动匹配并更新      |
| **日历视图**     | 月历展示，支持月份切换，含事件标记点                  |
| **待办事件列表** | 按日期展示待办事项，显示倒计时，支持勾选完成/手动删除 |
| **紧急提醒**     | 到期/即将到期事件以红色闪烁动画醒目提示               |
| **语音播报反馈** | 操作结果通过 TTS 语音播报（"操作成功"/"操作失败"）    |
| **小狗形象交互** | 三种状态 GIF（休眠/唤醒/分析中），配合对话气泡显示    |

### 1.2 操作方式

#### 语音操作流程

1. **打开语音开关**：点击界面上的语音交互开关
2. **唤醒系统**：说出关键词 **"重启"**，小狗会被唤醒并回应"我在"
3. **说出指令**：唤醒后 15 秒内说出你的事件指令

| 操作类型 | 语音指令示例                       | 系统效果                   |
| -------- | ---------------------------------- | -------------------------- |
| 添加事件 | "3月15日开会"      | 自动创建事件，添加到日历   |
| 删除事件 | "删除3月15日的开会"                | 匹配事件后标记为已完成     |
| 修改事件 | "把3月15日的开会改成3月20日的培训" | 匹配旧事件后更新名称和时间 |

> **注意**：删除和修改操作需要大模型返回的事件名称和时间**同时匹配**现有记录，才会执行。匹配失败时会播报"操作失败"。

#### 手动操作

- 点击右上角 **"+"** 按钮手动添加事件
- 点击日历日期查看该天的待办事件
- 勾选复选框标记事件完成/取消完成
- 点击 **✕** 按钮删除事件

### 1.3 达到的效果

- **智能意图识别**：基于阿里云百炼大模型（Qwen）自动解析用户语音中的事件意图
- **实时语音转写**：基于讯飞实时语音转写服务，边说边识别
- **交互反馈**：小狗动画 + 聊天气泡 + 语音播报，增加交互反馈

## 二、技术架构

### 2.1 技术栈

| 层级               | 技术                         | 用途                                  |
| ------------------ | ---------------------------- | ------------------------------------- |
| **桌面框架**       | Electron 38                  | 跨平台桌面应用容器                    |
| **前端**           | 原生 HTML/CSS/JS             | 无框架依赖，纯原生开发                |
| **语音识别 (ASR)** | 讯飞实时语音转写 API         | 麦克风 → 实时文字转写，WebSocket 通信 |
| **语义理解 (NLP)** | 阿里云百炼 Qwen3.6-Flash     | 将自然语言解析为日历操作指令          |
| **语音合成 (TTS)** | Web Speech API               | 浏览器内置语音合成，播报操作结果      |

### 2.2 项目结构

```
SmartCal/
├── main.js                  # Electron 主进程入口
├── preload.js               # 预加载脚本（安全 IPC 桥接）
├── package.json             # 项目依赖与脚本配置
├── assets/                  # 静态资源
│   ├── 唤醒等待.gif          # 小狗休眠动画
│   ├── 已唤醒.gif            # 小狗唤醒动画
│   ├── 分析.gif              # 小狗分析中动画
│   └── miyao.txt            # API 密钥配置文件
├── src/                     # 前端源码
    ├── index.html           # 主界面
    ├── style.css            # 样式表
    ├── app.js               # 应用初始化入口
    ├── calendar.js          # 日历组件
    ├── events.js            # 事件管理模块
    ├── voice.js             # 语音交互管理模块
    ├── settings.js          # 设置面板模块
    ├── bailian.js           # 阿里云百炼 AI 集成（Node.js 模块）
    └── iflytek-asr.js       # 讯飞实时语音转写集成（Node.js 模块）

```

### 2.3 数据流

```
用户语音 → 麦克风 → 讯飞ASR(WebSocket) → 实时文字
                                              ↓
                              唤醒词检测 → 小狗唤醒 → 继续监听
                                              ↓
                              用户说完话(2秒静音) → 百炼AI解析意图
                                              ↓
                              add/delete/modify → IPC → 主进程操作JSON文件
                                              ↓
                              TTS语音播报("操作成功"/"操作失败")
```

## 三、部署指南

### 3.1 前置条件

1. **操作系统**：Windows 11
2. **Node.js**：版本 18 或以上
3. **网络连接**：需要能访问阿里云百炼 API 和讯飞 API

### 3.2 第一步：安装 Node.js

1. 打开浏览器，访问 [https://nodejs.org](https://nodejs.org)
2. 下载 **LTS 版本**（长期支持版），推荐 18.x 或 20.x
3. 双击安装包，一路点"下一步"完成安装
4. 安装完成后，按 `Win + R`，输入 `cmd` 回车，在命令行中验证：

```bash
node -v     # 应显示 v18.x.x 或 v20.x.x
npm -v      # 应显示 9.x.x 或 10.x.x
```

### 3.3 第二步：获取源代码

将 `SmartCal` 整个项目文件夹放到你的电脑上，例如：

```
E:\yingyong\develop\SmartCal
```

### 3.4 第三步：安装依赖

打开命令行（PowerShell 或 CMD），进入项目目录：

```bash
cd E:\yingyong\develop\SmartCal
```

安装项目依赖（由于 Electron 较大，可能需要几分钟，请耐心等待）：

```bash
npm install
```

> **提示**：如果下载 Electron 很慢，项目已配置了国内镜像 `npmmirror.com`，会自动加速。

### 3.5 第四步：配置 API 密钥

编辑 `assets/miyao.txt` 文件，填入你的 API 密钥。文件格式如下：

```
APPID:你的讯飞APPID
APIKey:你的讯飞APIKey
APISecret:你的讯飞APISecret
ALI-APIKEY:你的阿里云百炼APIKey
```

### 3.6 第五步：启动应用

在项目目录下执行：

```bash
npm start
```
